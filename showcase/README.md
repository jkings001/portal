# Portal Showcase - Demonstração Interativa

Portal de demonstração com tema Aurora Profunda, mostrando os módulos principais do sistema de atendimento de TI.

## Módulos

- 📊 **Dashboard** - Gráficos e estatísticas de chamados
- 🎧 **Help Desk 24/7** - Sistema de tickets
- 📚 **Gestão de Treinamentos** - Plataforma de cursos
- 💬 **Integração Teams** - Sincronização com Microsoft Teams
- 📈 **Relatórios Avançados** - Análise de dados
- 🏢 **Gestão Multi-Empresa** - Controle centralizado

## Desenvolvimento

```bash
cd showcase
pnpm install
pnpm dev
```

## Deploy

Este projeto é deployado no Railway com domínio `showcase.jkings.team`

## Tema

Aurora Profunda - Fundo animado com ondas sinusoidais, nebulosas e partículas flutuantes.
